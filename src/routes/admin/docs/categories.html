Managing categories

Creating categories

On the admin sidebar there is a link to the categories page, from there you can click on the button saying New category

you wil be redirected to a page with an empty form the following fields should be entered for eavch category:

  • Name
  • Description
  • Image

After filling in the form click on the button Create

this will create a new category with the information provided

Editing a category

On the categories page there is a sidebar with a list of all the categories, click on one to edit that category

There are two tabs in the edit category page, the first one is for general information, like mame description etc..., the second one is to reorder packages in this category, simply drag the packages to the roder you want them in

on the bottom of the edit page there is a button called delete, this is how you delete a category, you should be very careful because there is no way to undo this
